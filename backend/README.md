Основные команды для проекта:

    - npm install //в папке backend 

    - npm start  //запуск сервера

    - node src/client.js  //подключение клиента

Для prisma:

    В backend/prisma
    - npx prisma generate
    В backend/
    - npm start  //запуск сервера
    
Выбранные технологии:

prisma orm, jwt, axios, dotenv, express.js, jest-test framework
